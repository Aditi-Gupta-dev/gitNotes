
# Git Notes

**First Time** 

Initialise git
```bash
git init
``` 

Add to staging area
```bash
git add .
``` 

Commit to final stage with a message
```bash
git commit -m "Message here"
``` 

Change branch name
```bash
git branch -M main
``` 

Link remote repository
```bash
git remote add origin <GithubLink>
``` 

Push to remote repository
```bash
git push -u origin main
```
